# Kuiz インストール手順

オンライン早押しクイズゲーム「Kuiz」をインストールする手順です。

## ?? 必要なもの
- Windows 10 (Build 19041以降) または Windows 11
- インターネット接続（ゲームプレイ時のみ）

## ?? インストール手順

### 超シンプル！3ステップで完了

#### ステップ 1: インストーラーを実行

1. `KuizSetup-1.0.0.exe` をダブルクリック
2. Windows Defenderの警告が出た場合：
   - 「詳細情報」をクリック
   - 「実行」をクリック

#### ステップ 2: インストールウィザードに従う

1. 「次へ」をクリック
2. インストール先を選択（デフォルトでOK）
3. 「インストール」をクリック
4. 完了！

#### ステップ 3: .NET 10 Runtime（必要な場合のみ）

インストーラーが自動で.NET 10の有無をチェックします。

**必要な場合**:
1. 自動的にダウンロードページが開きます
2. 「.NET Desktop Runtime 10.x.x」をダウンロード
3. インストール
4. 再度Kuizインストーラーを実行

## ?? 起動方法

**方法1**: スタートメニューから「Kuiz」を検索  
**方法2**: デスクトップのショートカットをダブルクリック（作成した場合）

## ?? 遊び方

### ホストとして遊ぶ
1. 「ルーム作成」をクリック
2. ロビーコードが表示されます（例: ABC123）
3. 友達にコードを教える
4. 「ゲーム開始」をクリック

### ゲストとして参加
1. 「ルームに参加」をクリック
2. 友達から教えてもらったロビーコードを入力
3. プレイヤー名を入力（デフォルト: ちびすけ明太子）
4. 「参加」をクリック

---

## ?? トラブルシューティング

### Windows Defenderが実行をブロックする

**原因**: 新しいEXEファイルのため
**解決策**:
1. 「詳細情報」をクリック
2. 「実行」をクリック

### アプリが起動しない

**解決策**: .NET 10 Desktop Runtimeをインストール
1. https://dotnet.microsoft.com/download/dotnet/10.0 を開く
2. 「.NET Desktop Runtime 10.x.x」をダウンロード
3. インストール後、Kuizを起動

### サーバーに接続できない

**解決策**: インターネット接続を確認してください

### 「問題の読み込みに失敗」エラー

**解決策**: 
1. インターネット接続を確認
2. しばらく待ってから再試行
3. Railway.appサーバーが起動中の可能性があります

## ??? アンインストール方法

### 方法1: 設定から
```
設定 → アプリ → インストールされているアプリ → Kuiz → アンインストール
```

### 方法2: コントロールパネルから
```
コントロールパネル → プログラムと機能 → Kuiz → アンインストール
```

**注意**: ユーザーデータ（プロフィール・ログ）は削除されません。
完全に削除したい場合は、以下のフォルダも手動で削除してください：
```
%LocalAppData%\Kuiz\
```
